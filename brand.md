# Brand — Cred

_Status: set (defined directly by project spec, not via brand-design wizard)_

## Palette
- Primary: `#7F77DD` (purple) — CTAs, accents, active states, links
- Success / live: `#1FA463` (green)
- Neutral text: `#171717` (foreground), `#6B7280` (muted), `#9CA3AF` (placeholder/disabled)
- Surface: `#FFFFFF` cards on `#FFFFFF` page background
- Borders: `#E5E7EB` at 0.5px

## Typography
- System sans stack (`-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, Helvetica, Arial`)
- Geist (already wired via `next/font/local`) available for headings if needed

## Shape & surface language
- Flat, minimal. No gradients, no drop shadows.
- Cards: white background, 0.5px border (`#E5E7EB`), 12px border radius (`.card` utility in `globals.css`)
- Badges: pill-shaped, green/purple/gray for live/private/mock
- Buttons: 12px radius, solid purple fill, no shadow

## Iconography
- Tabler icons, outline variant only (`@tabler/icons-react`)

## Voice
- Direct, confident, short. "Earn while you engage. Engage to be seen."
- No marketing fluff or exclamation points.
